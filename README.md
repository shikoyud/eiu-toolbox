# EIU Toolbox

A small SvelteKit toolbox for Eastern International University students.

This project is based on **SGU Toolbox**.

For the main usage and setup instructions, please read the SGU Toolbox README first:

> [SGU Toolbox README](../sgu-toolbox/README.md)

## Warning

This version does not have full feature parity with SGU Toolbox.

Compared to SGU Toolbox, this project is currently missing:

- Complete available lecturer surveys in one click.
